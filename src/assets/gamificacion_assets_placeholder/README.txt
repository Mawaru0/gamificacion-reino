Assets placeholder para la web de gamificación.

Estructura sugerida:
- backgrounds/reino-exterior-placeholder.png: fondo temporal.
- frames/: recuadros editables como PNG transparente.
- icons/: íconos simples temporales.
- decorations/: adornos pequeños.
- teams/: emblemas temporales de equipos.

Puedes reemplazar cada archivo por uno más bonito manteniendo el mismo nombre.
Así el código de React no se rompe.
